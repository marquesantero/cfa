Architecture

Governed execution for AI agents and data systems. Formalize intent. Evaluate policy. Control execution. Project state. Track lifecycle. Audit everything.

Read the WhitepaperGet Started

StateSignature

Formal typed contract for every intent — domain, datasets, constraints, target layer. Immutable, hash-verifiable.

Policy Engine

Declarative policy rules: APPROVE, REPLAN, or BLOCK. Auto-interventions correct constraints before execution.

Audit Trail

SHA-256 hash chain — cryptographically verifiable. Every decision, every fault, every remediation, permanently recorded.

CLI + MCP

cfa evaluate from the terminal. MCP server exposes governance to ChatGPT, Claude, Copilot, and any AI agent.

Rich Reports

Self-contained HTML reports with Chart.js — execution timeline, audit chain, lifecycle dashboard, compliance summary.

Lifecycle Indices

IFo, IFs, IFg, IDI — quantitative pipeline health. Promotion, watchlist, demotion based on evidence, not intuition.

Zero Runtime Dependencies

The entire CFA core runs on Python 3.11+ standard library. No mandatory pip installs. Optional extras: YAML, OpenTelemetry, MCP protocol.

pip install cfa-kernel && cfa evaluate "Join NFe with Clientes persist Silver"
